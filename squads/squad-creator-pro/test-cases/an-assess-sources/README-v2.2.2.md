## 🎯 Regra Especial v2.2.2: EVOLUÇÃO

**O que é evolução:**
- Comparação com OBRA ANTERIOR do mesmo autor
- Não é evolução "interna" dentro de um livro
- É mudança de posição/pensamento entre obras

**Aplicação Mecânica (não interpretativa):**

**$100M Offers → evolucao=FALSE**
- É a PRIMEIRA obra conhecida publicada
- Sem baseline anterior para comparar
- Regra binária: "primeira obra do autor = sempre FALSE"
- Impacto: -1 checkpoint em Atualidade (3/5 vs potencial 4/5)

**$100M Leads → evolucao=TRUE**
- Ofertas focava: "o que vender" (estrutura de oferta)
- Leads focam: "para quem vender" (geração de demanda)
- Mudança visível e documentada

**$100M Money Models → evolucao=TRUE**
- Leads tinha sequência linear
- Models focam dinâmica de sequência (paradigm shift)
- Mudança clara na abordagem de monetização
