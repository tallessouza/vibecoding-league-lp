# Fonts

## TASA Orbiter Display (DS-ADAPT-1.1)

Este diretório deve conter o arquivo de fonte:

- `TASAOrbiterDisplay-ExtraBold.woff2` (font-weight: 800)

### Download

1. Acesse https://fonts.google.com/specimen/TASA+Orbiter+Display
2. Baixe o arquivo e converta para .woff2 se necessário
3. Renomeie para `TASAOrbiterDisplay-ExtraBold.woff2`
4. Coloque neste diretório

### Licença

Verificar termos em https://fonts.google.com/specimen/TASA+Orbiter+Display/about

O `@font-face` já está configurado em `src/app/globals.css` apontando para `/fonts/TASAOrbiterDisplay-ExtraBold.woff2`.
