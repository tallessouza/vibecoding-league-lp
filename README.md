# Vibecoding League — Landing Page

Landing page para a plataforma de **Vibecoding Competitivo**: onde desenvolvedores competem em desafios de código com acompanhamento em tempo real.

## Stack

- **Next.js 14** — App Router + TypeScript
- **Tailwind CSS** — Estilização utilitária
- **Framer Motion** — Animações e transições
- **next-themes** — Suporte a tema claro/escuro
- **Radix UI** — Componentes acessíveis

## Desenvolvimento Local

```bash
pnpm install
pnpm dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Deploy

Projeto configurado com Vercel. Deploy automático na branch `main`.
